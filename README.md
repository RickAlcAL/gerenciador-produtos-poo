# gerenciador-produtos-poo
istema de linha de comando em Java focado na gestão de precificação de produtos (comuns, usados e importados). O projeto aplica conceitos fundamentais de Programação Orientada a Objetos, como herança estruturada, encapsulamento rígido e polimorfismo dinâmico na sobrescrita de métodos, além de validação ad-hoc via Enums.
